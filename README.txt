
Joda-Time Contributions area - Internationalization
===================================================
Joda-Time is a date and time library that vastly improves on the JDK.
This separate project provides additional support for internationalization.

See the home page for more details:
http://joda-time.sourceforge.net/contrib/i18n/index.html

The source code is held primarily at GitHub:
https://github.com/JodaOrg/joda-time-i18n
